Week 12: Methods (Part 2) 第 12 周：方法（下）

Step-by-Step Guide 分步指南

1. The Math Class

Java provides a Math class in the java.lang package containing static methods for common mathematical operations.

2. Common Math Methods

Math.pow(a, b) calculates a to the power of b.
Math.sqrt(a) finds the square root.
Math.random() generates a random double between 0.0 and 1.0.

1. Math 类

Java 在 java.lang 包中提供了一个 Math 类，其中包含用于常见数学运算的静态方法。

2. 常用 Math 方法

Math.pow(a, b) 计算 a 的 b 次幂。
Math.sqrt(a) 计算平方根。
Math.random() 生成一个介于 0.0 和 1.0 之间的随机 double 值。

Concept Visualization 概念可视化

Tips 提示

Java library functions save you time - no need to reinvent the wheel! Remember that Math.random() returns a double from 0.0 up to (but not including) 1.0.

Java 库函数能节省你的时间——无需重复造轮子！请记住，Math.random() 返回一个从 0.0 到（但不包含）1.0 的 double 值。
