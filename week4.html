Week 4: Java Environment 第 4 周：Java 环境与程序结构

Step-by-Step Guide 分步指南

Step 1: Overview of Java

Java is an object-oriented, platform-independent language. The slogan "Write Once, Run Anywhere" means compiled Java code can run on any device equipped with a Java Virtual Machine (JVM).

Step 2: Language Elements

Every Java application must have at least one Class and one main method. The main method is the entry point where the program starts executing.

Step 3: The JDK & IDEs

The JDK (Java Development Kit) contains the compiler (`javac`) and the JVM (`java`). Programmers use IDEs (like Eclipse or IntelliJ) to make writing, compiling, and running code easier.

步骤 1：Java 概述

Java 是一种面向对象的、跨平台的语言。其口号 “一次编写，到处运行” (Write Once, Run Anywhere) 意味着编译后的 Java 代码可以在任何配备了 Java 虚拟机 (JVM) 的设备上运行。

步骤 2：语言元素

每个 Java 应用程序必须至少有一个 类 (Class) 和一个 主方法 (main method)。主方法是程序开始执行的入口点。

步骤 3：JDK 和 IDE

JDK (Java Development Kit) 包含编译器 (`javac`) 和 JVM (`java`)。程序员使用 IDE（如 Eclipse 或 IntelliJ）使编写、编译和运行代码更容易。

Tips 提示

  • "Write Once, Run Anywhere": You will definitely be tested on why Java is platform-independent. The secret is the JVM! The compiler doesn't compile to machine code, it compiles to bytecode.
  • Class naming rule: The name of the public class MUST match the filename exactly (e.g., `public class Main` must be in `Main.java`).
  • 一次编写，到处运行：你肯定会考到为什么 Java 具有平台独立性。秘诀就在于 JVM！编译器不编译为机器码，而是编译为字节码。
  • 类命名规则：公共类的名称必须与文件名完全匹配（例如：`public class Main` 必须在 `Main.java` 中）。
