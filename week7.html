Week 7: Program Input and Output 第 7 周：程序输入与输出

Step-by-Step Guide 分步指南

Step 1: Executable Statements

Most lines in a Java method are executable statements—they perform actions like assigning variables, doing math, or calling methods. Every executable statement must end with a semicolon (;).

Step 2: Getting User Input (Scanner)

To make a program interactive, we read keyboard input. Java provides the Scanner class (in java.util) for this. You create a Scanner object and use methods like nextInt() or nextLine() to capture user typing.

Step 3: Formatting Numbers (printf)

Sometimes standard output looks ugly (e.g., 3.3333333). We use System.out.printf() with format specifiers like %.2f to format decimal numbers neatly (e.g., to 2 decimal places: 3.33).

步骤 1: Scanner 类接收输入

要从控制台读取数据，我们需要导入并使用 java.util.Scanner。例如：
Scanner input = new Scanner(System.in);
int val = input.nextInt();

步骤 2: 标准输出

Java 提供以下输出方法：
  • System.out.print(): 打印后不换行
  • System.out.println(): 打印后自动换行
  • System.out.printf(): 格式化输出

步骤 3: 格式化数字

使用 System.out.printf() 格式化输出：
System.out.printf("%.2f", 3.14159); 将显示 3.14。

Concept Visualization 概念可视化

Tips 提示

  • The Scanner Bug: If you use `nextInt()` and immediately follow it with `nextLine()`, the program will skip your string input! This is because `nextInt()` leaves the "Enter" key in the buffer. Always add an extra `scan.nextLine();` to clear the buffer.
  • Don't forget the import: `Scanner` requires `import java.util.Scanner;` at the very top of your file.
  • Scanner 导入： 绝不要忘记在代码最顶部加上 import java.util.Scanner;，否则编译器会报错说找不到该类！
  • printf 转换符： 记住，在 printf 中，%d 用于整数，%f 用于浮点数，%s 用于字符串。
