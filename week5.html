Week 5: Variables & Data Types 第 5 周：变量与数据类型

Step-by-Step Guide 分步指南

Step 1: Variables

A variable is a named memory location used to store data. In Java, you must declare a variable's data type before using it (e.g., int age = 20;).

Step 2: Primitive Data Types

Java has 8 primitive data types, but the most common are:

  • int: Whole numbers (e.g., 5, -10)
  • double: Decimal numbers (e.g., 3.14)
  • char: Single character in single quotes (e.g., 'A')
  • boolean: True or False

Step 3: The String Class

String is not a primitive; it is a Class (object). It stores text inside double quotes (e.g., "Hello"). You can use the + operator to concatenate (join) strings.

步骤 1: 什么是变量？

变量是内存中的一个命名存储位置，其值在程序运行期间可以改变。在 Java 中，所有变量在使用前必须先声明其类型。

步骤 2: 标识符规则

标识符（变量、类和方法名）可以包含字母、数字、下划线 (_) 和美元符号 ($)。但不能以数字开头，也不能是 Java 关键字。

步骤 3: 基本数据类型

Java 有 8 种基本数据类型，常用的包括：
  • int：整数
  • double：浮点数/小数
  • char：单个字符
  • boolean：真或假 (true/false)

Concept Visualization 概念可视化

Tips 提示

  • String is an Object: Notice how `String` starts with a capital 'S', but primitives like `int` and `double` are lowercase. This is a common trap!
  • Character vs String: `char` uses single quotes (`'A'`), while `String` uses double quotes (`"A"`). Mixing them up will cause a syntax error!
  • 命名规范：变量和方法名使用驼峰命名法（例如 studentAge），类名首字母大写（例如 StudentClass）。
  • 关键字限制：记住 class、public、static 和 void 等是 Java 保留关键字，绝不能用作变量名！
