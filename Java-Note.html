Fundamentals of Computer Systems

1.1 Overview of Computer and Programming

+

What is a Computer?

An electronic device that accepts data (input), processes it per instructions, and produces output.

IPO Cycle: Input → Processing → Output

Characteristics

  • Speed · Accuracy · Storage · Automation · Versatility

Programming Languages

  • Machine: Binary (0s,1s)
  • Assembly: Mnemonics
  • High-Level: Python, Java, C++

什么是计算机？

接受数据、按指令处理并产生输出的电子设备。

IPO循环：输入→处理→输出

1.2 Electronic Computer: Then and Now

+

Five Generations

1st (1940–56): Vacuum Tubes — ENIAC
2nd (1956–63): Transistors — IBM 7094
3rd (1964–71): ICs — IBM System/360
4th (1971–now): Microprocessors — Intel 4004
5th (now+): AI & Quantum

五个世代

第一代真空管→第二代晶体管→第三代IC→第四代微处理器→第五代AI/量子

1.3–1.4 Hardware & Software

+
  • Hardware: Input/Output, CPU (ALU, CU, Registers), RAM/ROM, Storage
  • Software: System (OS, Drivers) vs Application
Von Neumann: Data & instructions share memory. Fetch→Decode→Execute.
  • 硬件：I/O、CPU（ALU、CU、寄存器）、RAM/ROM、存储
  • 软件：系统软件（OS）vs 应用软件

Introduction to the Java Environment

4.1 Overview of Java

+

Key Features

  • Platform Independent: "Write Once, Run Anywhere"
  • OOP: Classes, objects, inheritance
  • Robust & Secure
.java → javac → .class (bytecode) → JVM → Machine Code
.java → javac → .class（字节码）→ JVM → 机器码

4.2 Java Language Elements

+
public class Hello {
  public static void main(String[] args) {
    System.out.println("Hello!");
  }
}
  • File name = public class name + .java
  • camelCase for variables

文件名=公共类名+.java。驼峰命名。

4.3 Setting Up Java Environment

+
  • JVM: Executes bytecode
  • JRE: JVM + libraries
  • JDK: JRE + javac + tools
javac Hello.java → compile
java Hello → run

JVM执行字节码。JRE=JVM+库。JDK=JRE+编译器+工具。

8
Selection Structures

8.1–8.3 if, if-else, switch

+
if (cond) { } else if (cond2) { } else { }
switch(x) { case 1: ...; break; default: ...; }
Key: switch needs break to prevent fall-through. Use else if for mutually exclusive conditions.
switch需要break防穿透。互斥条件用else if。
9
Repetition Structures

9.1–9.3 while, for, loops

+
while (cond) { ...; update; }
for (init; cond; update) { ... }
while: unknown iterations | for: known iterations. Always update loop variable!
while: 未知迭代 | for: 已知迭代。确保更新循环变量！
