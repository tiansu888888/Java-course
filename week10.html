Week 10: Repetition (Part 2) 第 10 周：循环结构（下）

Step-by-Step Guide 分步指南

1. The for Loop

A for loop is used when the number of iterations is known. It contains initialization, condition, and update in a single line.

Syntax:
for (init; condition; update) { // Code }

2. Nested Loops

A loop inside another loop is called a nested loop. The inner loop executes completely for each iteration of the outer loop.

1. for 循环

当已知迭代次数时使用 for 循环。它在单行中包含初始化、条件和更新操作。

语法：
for (init; condition; update) { // 代码 }

2. 嵌套循环

一个循环嵌套在另一个循环内部称为嵌套循环。外层循环的每次迭代中，内层循环都会完整执行一次。

Concept Visualization 概念可视化

Tips 提示

Use nested loops when you need to process tables, grids, or 2D arrays (like rows and columns).

当需要处理表格、网格或二维数组（如行和列）时，使用嵌套循环。

Coding Practice 编程实践

Task 1: Write a for loop that prints 1 to 5. 任务 1: 编写一个 for 循环打印 1 到 5。

for(int i=1; i<=5; i++) { System.out.println(i); }

Task 2: Write a nested for loop where 'i' and 'j' go from 1 to 2. 任务 2: 编写嵌套 for 循环，其中 'i' 和 'j' 均从 1 运行到 2。

for(int i=1; i<=2; i++) { for(int j=1; j<=2; j++) { System.out.println(i + " " + j); } }

Task 3: Write the header of a for loop that counts down from 10 to 1. 任务 3: 编写从 10 倒数到 1 的 for 循环头部。
