Knowledge Check 知识自测

1. A variable declared inside a for loop is accessible outside of it. 在 for 循环内部声明的变量，可以在循环外部访问。

2. Which error occurs when the code compiles but produces the wrong output? 代码成功编译但输出了错误的结果时，会发生哪种错误？

3. What converts Java Source Code (.java) into Bytecode (.class)? 什么将 Java 源代码 (.java) 转换为字节码 (.class)？

4. Which tool runs the compiled Java Bytecode? 哪个工具用于运行编译好的 Java 字节码？

5. Missing a semicolon produces a... 漏掉分号会产生一个...

Fill in the Blanks 填空练习

A error happens when you violate the rules of the Java language (like missing a semicolon). 当你违反 Java 语言规则（如遗漏分号）时，会发生 错误。

A error causes the program to crash while it is running (like dividing by zero). 错误会导致程序在运行期间崩溃（如除以零）。

Coding Practice 编程实践

Task 1: Write a complete main method signature. 任务 1: 编写完整的主方法 (main method) 签名。

public static void main(String[] args) { }

Task 2: Write a line of code to import the Scanner class. 任务 2: 编写导入 Scanner 类的代码行。

import java.util.Scanner;

Task 3: Instantiate a new Scanner object reading from System.in. 任务 3: 实例化一个新的、读取 System.in 的 Scanner 对象。

Scanner input = new Scanner(System.in);
