Week 3: Algorithm Design & Representation 第 3 周：算法设计与表示

Step-by-Step Guide 分步指南

Step 1: What is Pseudocode?

Pseudocode (false code) is an artificial and informal language that helps programmers develop algorithms.

Step 2: Flowcharts

A flowchart is a visual representation of an algorithm using standard shapes (Oval = Start/End, Parallelogram = Input/Output, Diamond = Decision).

步骤 1：什么是伪代码？

伪代码（虚拟代码）是一种人工的、非正式的语言，帮助程序员开发算法。

步骤 2：流程图

流程图是算法的视觉表示，使用标准形状（椭圆 = 开始/结束，平行四边形 = 输入/输出，菱形 = 决策）。

Concept Visualization 概念可视化

Tips 提示

  • Algorithm Representation: Pseudocode should be easy to read (like English), while Flowcharts should be easy to see visually. Know when to use which.
  • Exam Focus: Practice drawing full flowcharts and converting them to pseudocode, as this is a common 10-mark structured question!
  • 算法表示：伪代码应该易于阅读（类似于英语），而流程图应该易于从视觉上理解。知道何时使用哪一个。
  • 考试重点：练习绘制完整的流程图并将其转换为伪代码，因为这是一个常见的 10 分结构化大题！
