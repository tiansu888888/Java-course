Week 6: Operators and Expressions 第 6 周：运算符与表达式

Step-by-Step Guide 分步指南

Step 1: Arithmetic Operators

Standard math applies in Java: +, -, *, and /. There is also the Modulo operator %, which returns the remainder of a division.

Step 2: Relational Operators

These compare values and always result in a boolean (true/false). Examples: == (equal), != (not equal), > (greater than), <= (less than or equal to).

Step 3: Logical Operators

Used to combine multiple relational expressions: && (Logical AND), || (Logical OR), and ! (Logical NOT).

步骤 1: 算术运算符

Java 支持标准的算术运算：+（加）、-（减）、*（乘）、/（除）和 %（取模/求余数）。注意，整数除法（如 5 / 2）会截断小数部分得到 2。

步骤 2: 运算符优先级

Java 表达式遵循数学优先级规则：乘、除、取模先算，加、减后算。可以使用圆括号 () 来强制改变运算顺序。

步骤 3: 关系与逻辑运算符

关系运算符（>、<、==、!=）用于比较，结果为布尔值。逻辑运算符包括 &&（逻辑与）、||（逻辑或）和 !（逻辑非）。

Concept Visualization 概念可视化

Tips 提示

  • Operator Precedence: Remember BODMAS! Multiplication/Division always evaluate before Addition/Subtraction. Use parentheses () to force the order you want.
  • Equals vs Assignment: A single = assigns a value to a variable. A double == compares two values! Don't mix them up.
  • 取模运算符： x % y 计算的是 x 除以 y 的余数。例如 7 % 3 结果为 1。
  • 整数除法陷阱： 在 Java 中，5.0 / 2 得到 2.5，但 5 / 2 得到 2。如果要保留小数，其中一个操作数必须是小数类型！
