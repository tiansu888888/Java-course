Week 2: Introduction to Problem Solving 第 2 周：问题求解与算法

Step-by-Step Guide 分步指南

Step 1: The Software Development Method

Solving a problem with a computer requires a systematic approach:

  1. Specification: Understand the problem.
  2. Analysis: Identify inputs and outputs.
  3. Design: Create an algorithm (blueprint).
  4. Implementation: Translate into Java code.
  5. Testing: Find and fix bugs.
  6. Maintenance: Update software over time.

Step 2: What is an Algorithm?

An algorithm is a precise sequence of instructions for solving a problem.

步骤 1：软件开发方法

使用计算机解决问题需要系统的方法：
  1. 需求规范：理解问题。
  2. 分析：确定输入和输出。
  3. 设计：创建算法（蓝图）。
  4. 实现：翻译成 Java 代码。
  5. 测试：发现并修复 bug。
  6. 维护：随着时间的推移更新软件。

步骤 2：什么是算法？

算法是解决问题的一系列精确指令步骤。

Tips 提示

  • Algorithm Design: Never skip the design phase. Writing pseudocode or drawing a flowchart first will save you hours of debugging later.
  • Exam Focus: You will frequently be asked to identify flowchart symbols (e.g., Diamond = Decision). Memorize them!
  • 算法设计：切勿跳过设计阶段。先写伪代码或画流程图会为你节省数小时的调试时间。
  • 考试重点：你经常会被要求识别流程图符号（例如：菱形 = 决策）。记住它们！
